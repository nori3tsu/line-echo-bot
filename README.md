# Echo Bot

An example LINE bot just to echo messages

## How to use

### Install deps

``` shell
$ yarn
```

### Run

``` shell
$ export CHANNEL_SECRET=YOUR_CHANNEL_SECRET
$ export CHANNEL_ACCESS_TOKEN=YOUR_CHANNEL_ACCESS_TOKEN
$ yarn start
```

## Webhook URL

http://localhost:3000/callback
